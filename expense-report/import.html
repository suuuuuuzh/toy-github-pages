从 Excel 导入报销单

上传报销 Excel，自动生成 data.js；全部在你自己的浏览器里处理，不会上传到任何服务器

第 1 步：上传报销 Excel

支持两种表格结构：① 公司模板（日期 / 费用摘要 / 费用明细按类目分列 / 发票内容 / 发票金额）； ② 简单表格（表头有"金额"和"摘要/说明"即可，"日期""发票内容"可选）——没按类目分列的行会根据 摘要和发票内容的关键词自动划分类目（交通费 / 差旅-交通 / 住宿费 / 餐费 / 汽车费用 / 快递费 / 办公用品 / 业务招待费 / 其他），自动划分的在预览里是琥珀色下拉框，不对可以直接改。 一个 Excel 里有多个 sheet（比如每次出差一个 sheet）也没关系，会全部一起解析。

第 2 步：选择这批发票文件（可多选）

只会读取文件名（尝试从文件名里识别金额，比如"...141.00元.pdf"），不会读取文件内容，也不会上传。 金额能唯一对应上的会自动选好，金额重复或没找到的下面可以手动选。
